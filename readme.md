# Chatter

Chatter is a simple chat application built using websockets, React, and es6.

### Dependencies

* Node JS
* Npm
* Bower
* Babel (6 to 5)

### Quick start

Run `./setup.sh`

### Compile assets

Run `gulp`

### Start the server

Run `gulp server`

### Demo site

View the demo: https://chatter3.herokuapp.com/
